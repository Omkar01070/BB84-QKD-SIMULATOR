 BB84 Quantum Key Distribution Simulator

Overview

This project simulates the BB84 Quantum Key Distribution protocol with noise modeling and eavesdropper detection.

# How to Run

1. Clone the repository
2. Install dependencies
3. Run the application
4. Open browser at http://127.0.0.1:5500

 Installation

pip install -r requirements.txt

 Run

python app2.py

 Features

* QBER calculation
* CASCADE error correction
* Privacy amplification
* ML-based eavesdropper detection
* Web interface
